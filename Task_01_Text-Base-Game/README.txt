Task 1: Number Guessing Game (Java)

Description:
This is a console-based number guessing game developed using Java.
The program generates a random number between 1 and 100 and the user
has to guess the correct number. The program provides feedback for
each guess until the correct number is guessed.

Concepts Used:
- Java Basics
- Loops
- Conditional Statements
- Random Class
- Scanner Class

How to Run:
1. Compile the file using: javac Task1_NumberGuessingGame.java
2. Run using: java Task1_NumberGuessingGame
